- HTML / CSS / JavaScript (Vanilla)

## Déploiement GitHub Pages
1. Push ce dossier sur un repo GitHub
2. Settings → Pages → Source : `main` / `root`
3. Ton site sera live sur `https://TON_USERNAME.github.io/NOM_REPO`
